# Own Pakage

Installer
```bash
apt update
apt upgrade -y
pkg install wget
wget https://github.com/termuxandlinux/pkg/releases/download/V.1.0/pkgo_1.0_all.deb
dpkg -i pkgo_1.0_all.deb
```

For Installing Pakage
```bash
pkgo {pakage}
```

The pkg found you in the repository
